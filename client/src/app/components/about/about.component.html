About Quantity Measurement App

A fullstack web application built for seamless quantity conversion and comparison.

Project Overview

The Quantity Measurement App is a robust, modern full-stack application that provides users with an intuitive interface to convert and compare different physical quantities. It supports four main types of measurements: Length, Volume, Weight, and Temperature.

Developed during the Bridgelabz training program, this project demonstrates a complete end-to-end implementation including secure user authentication, role-based access, and a responsive user interface.

Key API Endpoints

  • POST /api/Auth/login - Authenticate user and get JWT token
  • POST /api/Auth/register - Register a new user
  • GET /api/Quantity/types - Get all supported quantity types
  • POST /api/Quantity/convert - Convert between units
